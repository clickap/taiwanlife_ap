ewant admin is based on the great tool click-ap.adminer (www.click-ap.com).
The main advantage of this plugin is, it can handle different types of ewant administrator(www.ewant.org).
So it works with MySQL(ewant2).
This plug-in handle by Elaine (elaine@click-ap.com) now.

Installation:
To install ewant admin just copy the folder "ewantadmin" into your moodle/local/ewantadmin.
After that you have to go to http://your-moodle/admin (Site administration -> Notifications) to trigger the installation process.

Using:
To use Moodle Adminer go to "Site administration" -> "ewant admin".
